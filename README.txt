《流浪地球》读书分享兼容版交付包

特点：
- 背景音乐同时提供 MP3 和 AAC，页面优先使用 MP3
- 所有实际使用到的视频统一转为更通用的 H.264 + AAC MP4
- 只保留最终演示所需文件，方便拷贝到其他电脑

使用方法：
1. 双击 start-video-deck-grade3.bat
2. 或直接打开 video-deck-grade3.html
3. 建议使用 Edge 或 Chrome 全屏播放

包含内容：
- video-deck-grade3.html
- start-video-deck-grade3.bat
- assets/audio
- assets/images
- assets/video
